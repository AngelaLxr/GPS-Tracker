# GPS Tracker 
Uses a browser with google maps to more accuractely detect your location. 
Originally we were using a webdriver but browser made things easier and more accurate location tracking. 
This will ask for permission to track your location and once that is done, it will open up google maps and it should track where you are.

# Voice Activation
With the voice actication addition, you can now simply say "get my location" and it will automatically track your location.

# Added battery monitoring
Lets you know how much the battery is right now and when its below 20%. The code will make it so if you're battery is 20% or lower, it will trigger a pop up.
